Password Security policy
## minimum requirements
1. Minimum length: 12 characters
2. Atleast one uppercase
3. atleast one lowercase
4. atleast one number
5. atleast one special character
6. avoid common passwords

## security reasoning
### length requirements
having a longer password grows the number of possibilities your password can be, making it harder to be guessed
### character variety
using a range of different characters makes it harder to guess
### avoiding commmon passwords
attackers usually use attacks testing common passwords first
